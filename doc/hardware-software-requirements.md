# 3.2 Hardware/Software Requirements

| Component / Tool | Description | Price / Source |
|---|---|---|
| **Computer / Laptop** | Intel i5 or higher, 8GB RAM, 256GB SSD minimum | Personal or University Lab |
| **Operating System** | Windows 10 / 11 or macOS | Already installed |
| **IDE / Code Editor** | Visual Studio Code or similar | Free / Community Edition |
| **Programming Languages** | HTML, CSS, JavaScript, TypeScript | Free |
| **Frameworks / Libraries** | React.js / Next.js + TypeScript for front-end, Express + TypeScript for back-end | Free |
| **CSS Framework** | TailwindCSS | Free |
| **Database** | Supabase (PostgreSQL) | Free tier / Paid plans |
| **Authentication** | Supabase Auth + Firebase (Google OAuth) | Free tier |
| **Real-time Communication** | Socket.io | Free |
| **AI / Automation Tools** | n8n (workflow automation), DeepSeek API (LLM) | Free / API credits |
| **Version Control** | Git & GitHub | Free |
| **Web Browser for Testing** | Chrome / Firefox | Free |
| **Other Tools** | Figma / Canva for UI design | Free / Subscription optional |
